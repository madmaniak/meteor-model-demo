meteor-model-demo
=================

Run
---
<code>meteor</code>


Steps to reproduce:
---

0. meteor create <code>model-demo</code>
1. Copy <code>lib/</code> from [meteor-model](https://github.com/madmaniak/meteor-model).
2. <code>meteor add coffeescript</code>
3. Make models in <code>models/</code>
4. Set html.

Run it.
